<section class="py-20 md:py-28 px-4 bg-surface-1">
<div class="mx-auto max-w-5xl">
<p class="text-xl md:text-2xl text-fg-3 leading-snug tracking-tight max-w-2xl" data-scramble><span class="text-fg-1">Cases are the unit of work.</span> Every incoming request, claim, order, or ticket becomes a case. Each case carries its full lifecycle — conversations, files, decisions, and outcomes — in one structured view.</p>
<div class="mt-12 grid grid-cols-1 md:grid-cols-3 gap-4">
<div class="rounded-lg border border-line bg-surface-2 flex flex-col" style="padding: 1.25rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Structured Threads</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Every conversation, action, and decision lives in a thread attached to the case. Human messages and agent activities interleave in one timeline.</p>
</div>
<div class="rounded-lg border border-line bg-surface-2 flex flex-col" style="padding: 1.25rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Files and Artifacts</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Documents, extracted data, generated reports, and code outputs attach directly to the case. Every artifact is versioned and traceable to the step that produced it.</p>
</div>
<div class="rounded-lg border border-line bg-surface-2 flex flex-col" style="padding: 1.25rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Phase and Motion</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Cases move through lifecycle phases automatically as procedures complete. Motion state tracks whether work is active, waiting, or blocked — giving teams real-time visibility.</p>
</div>
</div>
</div>
</section>

<section class="py-20 md:py-28 px-4">
<div class="mx-auto max-w-5xl">
<p class="text-xl md:text-2xl text-fg-3 leading-snug tracking-tight max-w-2xl" data-scramble><span class="text-fg-1">Procedures are your playbooks.</span> Define multi-step workflows in natural language. Each procedure combines AI-assisted steps with deterministic controls, so execution is fast and auditable.</p>
<div class="mt-12 grid grid-cols-1 md:grid-cols-2 gap-6">
<div class="rounded-lg border border-line" style="padding: 1.5rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Natural Language Authoring</p>
<p class="text-sm text-fg-3 leading-relaxed mb-3">Write workflow instructions the way you explain them to a colleague. No flowchart editors, no BPMN diagrams. Domain experts own the logic directly.</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Add code blocks where precision matters — data transformations, API calls, format validation — and let natural language handle the rest.</p>
</div>
<div class="rounded-lg border border-line" style="padding: 1.5rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Stages and Branching</p>
<p class="text-sm text-fg-3 leading-relaxed mb-3">Break complex processes into stages. Each stage has its own instructions, validation criteria, and escalation rules.</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Branching logic routes cases based on extracted data, classification results, or human decisions — keeping execution deterministic where it needs to be.</p>
</div>
</div>
</div>
</section>

<section class="py-20 md:py-28 px-4 bg-surface-1">
<div class="mx-auto max-w-5xl">
<p class="text-xl md:text-2xl text-fg-3 leading-snug tracking-tight max-w-2xl" data-scramble><span class="text-fg-1">Agents that operate within boundaries.</span> Repetitive tasks run autonomously — extraction, triage, classification, follow-ups. But every agent action is bounded by the procedure that governs it.</p>
<div class="mt-12 grid grid-cols-1 md:grid-cols-3 gap-4">
<div class="rounded-lg border border-line bg-surface-2 flex flex-col" style="padding: 1.25rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Predictive Execution</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Agents anticipate next steps based on case context and procedure history. Routine cases flow end-to-end; edge cases escalate to the right person.</p>
</div>
<div class="rounded-lg border border-line bg-surface-2 flex flex-col" style="padding: 1.25rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Human-in-the-Loop</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Any step can require human approval, review, or override. The agent pauses, presents context, and waits — keeping the human in control where judgment matters.</p>
</div>
<div class="rounded-lg border border-line bg-surface-2 flex flex-col" style="padding: 1.25rem;">
<p class="text-sm font-medium text-fg-1 mb-2">Full Audit Trail</p>
<p class="text-sm text-fg-3 leading-relaxed mb-0">Every agent action, decision, and data access is logged. Compliance teams can trace any outcome back to its source — the instruction, the input, and the reasoning.</p>
</div>
</div>
</div>
</section>
