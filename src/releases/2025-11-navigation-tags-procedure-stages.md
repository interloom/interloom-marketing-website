**Workspace Navigation and File Management - November 11, 2025.** Spaces now appear directly in the sidebar for one-click workspace access. Each space also has a dedicated Files tab to browse, preview, and manage all related documents and attachments in one place.

**Tags - November 5, 2025.** Interloom now supports flexible tagging for task organization. Teams can apply custom tags, then filter and group work by the dimensions that matter most. A fast Tag Picker supports instant matching while typing, and the AI assistant can suggest and apply tags automatically based on task content to improve categorization consistency.

**Procedure Stages - November 25, 2025.** Procedures can now be modeled as sequential stages, making progress visible from intake through resolution. Teams get a shared view of current state and next steps, replacing ad-hoc status updates with a clear, structured progression.
