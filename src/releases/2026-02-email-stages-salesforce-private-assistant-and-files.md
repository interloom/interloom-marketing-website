**Email: Drafting, Sending, and Threading - February 2, 2026.** Interloom now supports the full email lifecycle in platform: inbound email creates tasks, agents draft responses from full case context, users review and send, and replies route back into the correct task thread through signed reply addresses.

**Stages - February 13, 2026.** Cases now move through structured phases with full configurability per procedure. Teams can create, edit, reorder, and remove stages, while managers can view pipeline position across active work at a glance.

**Salesforce Integration - February 16, 2026.** Agents can query and update Salesforce records directly from Interloom, reducing context switching between CRM and case operations while bringing account context into ongoing conversations.

**Private Assistant - February 17, 2026.** Users can now run private AI conversations outside case threads for ad-hoc research, drafting, and brainstorming, with separate history and navigation placement.

**Structured File Processing - February 22, 2026.** The assistant can parse and analyze uploaded files directly in conversation, including CSV, TSV, XLS, XLSX, JSON, JSONL, and PDF up to 100 MB. It returns structure-aware previews and supports sheet-level handling for spreadsheets.

**Comprehensive Activity Trail.** Every procedure action - stage transitions, tool executions, and agent decisions - is logged with timestamped history directly in the task view, enabling auditable, compliance-ready process documentation.
